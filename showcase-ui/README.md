# Showcase UI

Next.js frontend that demonstrates LLM Gateway capabilities with:

- interactive tutorial mode
- snippet playground with live outputs
- real chat interface
- gateway health/metrics panel

## Run

1. Copy `.env.example` to `.env.local`
2. Ensure gateway and sidecar are running
3. Install dependencies:

```bash
npm install
```

4. Start dev server:

```bash
npm run dev
```

Open http://localhost:3000.
